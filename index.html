THE DEEPEST DEPTHS

Welcome to the Deepest Depths — a dungeon crawler debt collector game where you have been thrown into the depths so you can gather gold to pay off the money that you owe. This is a game where you never gain levels, but the monsters do, and you have to use your smarts to defeat the denizens of the depths, find gold, weapons and potions. Reach the bottom to conquer the darkest overlord at level 20 and surface debt free!

CONTROLS

HOW THIS GAME WAS MADE

The Deepest Depths was built entirely using AI-assisted development. The game concept, design, and creative direction were done by Mirko Ruckels. The code, rendering engine, procedural map generation, and all game systems were developed collaboratively with Claude (Anthropic) as the AI coding assistant. All pixel art enemy sprites and visual assets were generated using Gemini (Google) and processed for the C64 aesthetic. The game runs entirely in the browser using HTML5 Canvas and the Web Audio API — no frameworks, no libraries, just vanilla JavaScript.
